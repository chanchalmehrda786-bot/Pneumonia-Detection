# 🩺 Pneumonia Detection using Deep Learning (CNN)

## 📌 Project Overview

This project is a **Deep Learning-based image classification system** that detects **Pneumonia** from Chest X-ray images.
A **Convolutional Neural Network (CNN)** model is trained to classify X-ray images into:

* ✅ Normal
* ❌ Pneumonia

The model helps in early detection of pneumonia, which can assist healthcare professionals in diagnosis.

---

## 📂 Dataset

The dataset used in this project is:
**Labeled Chest X-Ray Images (Pneumonia)** (Kaggle)

It contains:

* Training set
* Validation set
* Test set

Each set has two categories:

* NORMAL
* PNEUMONIA
